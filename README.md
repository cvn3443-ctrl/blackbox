# 🎲 BlackBox – لعبة الصندوق الأسود

منصة ألعاب ربحية تعتمد على اختيار الصناديق مع مستويات مخاطرة متعددة.  
تم تطويرها بواسطة **Mo.dark Engineering**.

---

## 🌟 الميزات الرئيسية

- 🎯 **3 مستويات مخاطرة**: منخفض، متوسط، مرتفع.
- 💰 **نظام إيداع وسحب** عبر Binance (USDT).
- 🌍 **دعم 4 لغات**: العربية، الإنجليزية، التركية، الفرنسية.
- 🛠️ **لوحة تحكم كاملة** لإدارة المعاملات.
- 📊 **إحصائيات لحظية** للمستخدمين والأرباح.
- 🔒 **آمن** مع حماية البيانات الحساسة.

---

## 🛠️ التقنيات المستخدمة

- **الخادم**: Node.js + Express
- **قاعدة البيانات**: SQLite
- **الواجهة**: HTML5 + CSS3 + JavaScript
- **الاستضافة**: Render (سحابة مجانية)
- **الإيداع**: Binance API
